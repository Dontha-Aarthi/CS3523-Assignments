To test the code, change the data in input.txt and then compile and run using the command mentioned below.

Command to compile the source code is:
	gcc Assgn1Src-CS20BTECH11015.c -lpthread -lm
	

And the command to run the generated executable is: 
	./a.out
	
After running the time taken for searching nearest point and the nearest point will be printed on the terminal.
